Exquisite Corpse

“The exquisite corpse will drink the new wine.” the game’s first sentence — Paris, 1925

Exquisite Corpse is a surrealist parlor game in which each participant takes turns writing or drawing on a sheet of paper, folding it to conceal the contribution, and passing it to the next player. This version seats a language model at the table. Each player adds a few words, and the sheet folds as it passes so the next player sees only the newest contribution. The complete poem appears when someone ends the game.
